## Command Line Notes

- To list files in the current directory:
  ```sh
  ls
  ```
- To display the contents of `hello_world.txt`:
  ```sh
  cat hello_world.txt
  ```
- To print "Hello, World!" in the terminal:
  ```sh
  echo "Hello, World!"
  ```
